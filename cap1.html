1. Dall’HTML 4 all’HTML 5

Le pagine web vengono create mediante un linguaggio di marcatura, l’HTML (HyperText Markup Language = linguaggio di marcatura degli ipertesti). L’HTML 4.01 divenne lo standard di specifiche per i documenti web nel 1999. HTML 5, la sua evoluzione, tiene conto della rapida crescita di banda a disposizione degli utenti e di conseguenza dell’accesso di diversi tipi di media al web (cellulari, tablet, smart tv). Le pagine web sono passate ad essere da semplici testi contornati da immagini a pagine ricche di contenuti multimediali (rich online experiences) fino ad arrivare a delle vere e proprie applicazioni web (web application o web app), delle pagine web con interfaccia utente simile a quella di un programma o una app di un sistema mobile (vedi pixlr e google drive, ad esempio).
Alcuni metodi di costruzione di pagine web sono diventati obsoleti come alcuni elementi presenti nell’HTML 4. È sconsigliato l’uso, ad esempio, di tutti gli elementi riferiti alla gestione del formato del testo e della pagina (colore, tipo di carattere, dimensione, spaziature e margini) a favore di una suddivisione di compiti tra HTML (che definisce i contenuti e la struttura della pagina)  e foglio di stile CSS (che ne definisce le caratteristiche grafiche, quindi la formattazione).
In questa guida si userà la sintassi HTML 5 corretta. Alcuni elementi nuovi di HTML 5 verranno visti successivamente causa la non completa compatibilità con tutti i browser e la necessità di mantenere le pagine funzionanti con browser meno aggiornati.
